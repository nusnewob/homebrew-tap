# homebrew-tap

A Homebrew Tap repository containing custom formulae.

## Setup

To add this tap to your Homebrew installation:

```bash
brew tap nusnewob/tap
```

## Available Formulae

- [`gopidof`](github.com/nusnewob/gopidof): A lightweight `pidof` implementation written in Go - a native Go alternative to the deprecated Homebrew pidof package.

## Installing Formulae

After tapping this repository, you can install any formula using:

```bash
brew install <formula-name>
```

## Uninstalling

```bash
brew uninstall <formula-name>
brew untap nusnewob/tap
```
